First Robotics: Charged Up (2023)

Strip Tracking || TOFs || The Autonomous Award

Overview

My primary focus for this project was developing an effective autonomous routine. Using **reflective strip distance tracking** and encoders, our robot reliably navigated the field to score cones and cubes. This work was critical in securing our team the **Autonomous Award**.

Our team's robot, designed and built for the Charged Up competition.

Features

  • **Advanced Autonomous Sequences:** Executed precise actions to pick up and score game pieces.
  • **Reflective Strip Distance Tracking:** Accurately positioned the robot for consistent performance.
  • **Award-Winning Performance:** Recognized with the prestigious Autonomous Award.

Video showcasing the robot's autonomous routine in action.

Reflection

This project taught me the value of rigorous testing and a clean, well-documented codebase. The experience solidified my skills in sensor integration and autonomous system design. Moving forward, I'd like to explore using computer vision for even greater precision.
